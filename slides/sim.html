Notes

Start by setting the true population slope to zero. In that case, there is no real relationship between x and y. If you run many repeated samples, only about the chosen alpha level should cross the red critical lines. For example, with alpha = 0.05, roughly 5% of samples should be called significant just by chance.

Then increase the slope. Now there is a real relationship in the population. If the sample size is large enough and the noise is not too strong, more simulated t-values should move beyond the red critical lines. This is the idea of statistical power: how often a test detects an effect when an effect is really there.

Remember what a significant result means. It means your sample t-value landed somewhere that would be unusual if the true slope were zero. It does not mean there is only a 5% chance that the null hypothesis is true. Alpha is the long-run false-positive rate of the decision rule.
